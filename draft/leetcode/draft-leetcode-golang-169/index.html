Leetcode Golang 169.Majority Element

Leetcode Golang 226.Majority Element

Jan 12, 2021

169 Majority Element

題目連結：https://leetcode.com/problems/majority-element/

解法

這題雖然是 easy 的題目，但解法卻非常特別，也因此這題目可以拿到這麼多人的 like。

一看到這道題目相信大部分人想到的都是用一個 hash table 去記錄所有數字出現的次數。

func majorityElement(nums []int) int {
  maxCount := 1
  res := nums[0]
  table := map[int]int{
    res: maxCount,
  }
  
  for i := 1; i < len(nums); i++ {
    n := nums[i]
    table[n]++
    if table[n] > maxCount {
      maxCount = table[n]
      res = n
    }
  }
  return res
}

但因為題目有強調 majority element 一定會出現總數的一半以上，所以就有一個特別的解法，這個 algorithm 叫做 Boyer–Moore majority vote algorithm。

演算法本身也蠻單純的，就是一直計算到目前為止出現過最多次的 element，如果下一個 element 跟現在的一樣，次數就 +1，如果不是就 -1，當次數小於 0 時，代表可能有其他的 element 才是 majority element。

func majorityElement(nums []int) int {
	res := nums[0]
	count := 1
	for i := 1; i < len(nums); i++ {
		if nums[i] == res {
			count++
		} else {
			count--
			if count == -1 {
				res = nums[i]
				count = 1
			}
		}
	}
	return res
}

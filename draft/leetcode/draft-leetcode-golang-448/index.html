448 Find All Numbers Disappeared in an Array

題目連結：https://leetcode.com/problems/find-all-numbers-disappeared-in-an-array/

解法

初次看到這個題目的直覺解法是用 hash table 記錄哪些數字有出現過，之後在用 for loop 跑過，看哪一個數字沒有出現，將他加入到回傳的 array 當中。

func findDisappearedNumbers(nums []int) []int {
  table := map[int]bool{}
  
  for _, n := range nums {
    table[n] = true
  }
  
  res := []int{}
  for i := 1; i <= len(nums); i++ {
    if _, ok := table[i]; !ok {
      res = append(res, i)
    }
  }
  return res
}

這樣的解法雖然 AC(accepted)，但題目最後有說

Could you do it without extra space and in O(n) runtime? You may assume the returned list does not count as extra space.

所以這個題目的重點是在如何將空間複雜度 O(n) 的解法優化成 O(1)。

因為題目限定數字的大小不會超過 array 的長度，所以只要將相對應的數字移動到相對應的位置就好，最後在跑過一次 for loop 將數字對不起來的欄位回傳就可以完成。

func findDisappearedNumbers(nums []int) []int {
	for i := 0; i < len(nums); i++ {
		n := nums[i]
		for nums[n-1] != n {
			nums[n-1], nums[i] = nums[i], nums[n-1]
			n = nums[i]
		}
	}

	res := []int{}
	for i := 0; i < len(nums); i++ {
		if nums[i] != i+1 {
			res = append(res, i+1)
		}
	}

	return res
}
